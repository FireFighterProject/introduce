# 🚨 재난 대응 통합 관리 시스템

## 📌 프로젝트 개요

소방 차량 및 출동 자원을 중앙에서 관리하기 위한 시스템이다.  
차량 등록, 출동 관리, 활동 추적, 통계 및 보고서 기능을 제공한다.

---

## 🧩 주요 기능

### 1. 메인
- 재난 / 평상시 상태 전환 (전역 상태)
- 상태에 따라 현황 집계 방식 변경

---

### 2. 차량 관리
- 차량 등록 (수동 입력 / 엑셀 업로드)
- 상태 관리: `대기 / 활동 / 철수`
- 자원 집결지 여부 토글
- GPS 문자 일괄 송신

---

### 3. 현황
- 차량 및 인원 자동 집계
- 재난 여부에 따라 집계 대상 변경
- 지역 / 상태 / 차종 기준 통계 제공

---

### 4. 출동 관리
- 대기 차량 선택 → 차량 편성 → 출동 명령 작성
- 출동 명령 문자 송신 (AVL 기반)
- 상태 흐름:

대기 → 확인중 → 성공


---

### 5. 활동 관리
- 활동 중 차량 리스트 조회
- 복귀 처리 (→ 대기 상태 변경)
- 장소 이동 (출동 정보 변경)
- 차량 위치 및 이동 경로 확인

---

### 6. 지도
- 활동 차량 GPS 기반 위치 표시
- 지역 클릭 / 드래그로 통계 조회
- 필터: 지역, 소방서, 차종

---

### 7. 통계
- 일시별 / 지역별 / 차종별 / 활동시간 등 통계 제공
- 로그 기반 자동 집계

---

### 8. 보고서
- 조건 설정 후 보고서 생성
- PDF / Excel 다운로드 지원

---

## 🧠 핵심 설계

### 상태 관리
- 차량 상태: `대기 / 활동 / 철수`
- 출동 상태: `DRAFT → SENT → ENDED`
- 상태 전이 검증 로직 적용

### GPS 처리
- AVL 단말기 → 서버 → DB 저장
- 현재 위치 / 이력 분리 관리
